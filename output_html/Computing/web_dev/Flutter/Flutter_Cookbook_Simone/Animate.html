Implicit Animation with AnimatedContainer

Its properties can change over a specific time period. The required property of the AnimatedContainer widget is duration. It specifies the time length of said animation. Animations are *not* bound to any other widget on the screen. In a class that includes a mixin class _ShapeAnimationState extends State with SingleTickerProviderStateMixin {`

mixin *is a class containing methods that can be used by other classes without inheriting from those other classes. Basically, we are including the class, not using it as a parent class* -Simon[book]
late AnimationController contro;
late Animation ani;

@override
void initState() {
  super.initState();
  contro= AnimationController(
    duration: const Duration(seconds: 3),
    vsync: this,
  );
  ani = Tween(begin:0,end:450)
    .animate(contro)
    ..addListener(() {
  	  moveBall();
  });
}

This will cause of the incrementing of a variable from 0-450 over 3 second period.

void moveBall() {
  setState(() {
    posTop = animationTop.value;
    posLeft = animationLeft.value;
});
}
