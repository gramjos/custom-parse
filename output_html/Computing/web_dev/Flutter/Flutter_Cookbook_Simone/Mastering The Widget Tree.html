Row and Column Widgets

  • A Row can be inserted into a Column and vice versa

Both have the properties CrossAxisAlignment and MainAxisAlignment - both these properties are abstractions of x y axis and tweaking these values will alter the layout. Notice, the MainAxisAligment for Column widgets is vertically down and Row is horizontal across. Vice versa for the CrossAxisAlignment parameter. !300 the only real difference between is their default orientation axis they start laying out children
