Modern Web Dev

The internet (large network (connection between 2+ computers)) is an information system enabling sharing.

Identifying Devices on the Network (internet)

IP (Internet Protocol) address are used to identify devices. IPv4, internet protocol version four and there is IPv6. The versioning refers to bit width used to created said IP. The larger the bit width the larger the number can be (allows for devices on the network).

Communicating between devices

Computers request resources from other computers resource all encompassing vague term on purpose.

Type of computer on the network

Has specialized, optimized functionality for its tasks. Server - supplies or forward resources to other computers Client - usually the computer that requests the resources

Domain Name System (DNS)

Like an address book for websites Takes an URL and returns the corresponding server IP address
