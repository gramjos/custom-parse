Jargon: higher order function

functions that take (as parameter) or return a function

Maintaining state
const updater = () => {
  let counter = 0;
  return () => {  
    counter += 1;
    console.log(counter);  
  }  
}  
const update = updater();
update();
update();
update();

Note, the scope referencing the variable counter. The function in the return statement is able to access the variable because it is 'child scope'. This type referencing is called a closure. For completeness, the function returns just a reference so....
