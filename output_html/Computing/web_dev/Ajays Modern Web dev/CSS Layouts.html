Jargon: Layout Viewport
  • The place where the browser arranges web components
  • Has a height and a width, measured in pixels (px) which is determined by the device.
Properties of the viewport
  • By default the elements are placed in the top left of the viewport, then are stacked below
  • the top left of the viewport is (0,0) of a coordinate plane
Pattern: Box Model

Describes how: - HTML elements are relatively positioned - elements contents are positioned within the element - properties of the elements border. In a basic setup, the following attributes are able to be manipulated. All can be override ('hacked') in there own way. !box_model.png

display mode

Changing the way how viewport is displays nested elements with block, inline, grid, and flex.

Jargon: flex box

An element becomes a flex box when display mode is set to flex.
