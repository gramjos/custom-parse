Listeners Anatomy

  1. target
  2. user action
  3. listener action
Target
  • area to listen for an action
  • can be an art board or nested art bard or object within art board
Defining an art board

While the listener is selected, select the target button and then either select the object in the hierarchy or directly on the stage. Once the defined action happens within the target area the listener will react.

Types of Targets
  • Shape or group will create a hit box
  • Art board or nested art boards will listen for any events coming from those boards
